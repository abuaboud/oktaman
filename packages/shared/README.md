# @oktaman/shared

Shared utilities, types, and configurations for the OktaMan monorepo.

## Usage

Import from other packages in the monorepo:

```typescript
import { add, subtract, User, Status } from '@oktaman/shared';
```

## Development

```bash
# Build the package
npm run build

# Watch mode for development
npm run dev

# Clean build artifacts
npm run clean
```
