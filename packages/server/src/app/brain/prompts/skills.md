## Usage

1. **Recognize**: Check if user's task matches a skill's domain
2. **Read**: Use `cat {skill.path}` to load the full SKILL.md with instructions
3. **Execute**: Follow the skill's step-by-step workflow
4. **Access files**: Skills may include supporting files in their directory

Always read the full skill before using it.
