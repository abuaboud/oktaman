# @oktaman/ui

UI component library built with [shadcn/ui](https://ui.shadcn.com/) in New York style.

## Setup

This package uses:
- **shadcn/ui** - Re-usable components built with Radix UI and Tailwind CSS
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Type safety
- **New York Style** - shadcn's New York theme variant

## Adding Components

To add shadcn components to this package:

```bash
cd packages/ui
npx shadcn@latest add [component-name]
```

## Usage

In your app, install this package and import components:

```tsx
import { Button } from "@oktaman/ui"
import "@oktaman/ui/styles.css"
```
